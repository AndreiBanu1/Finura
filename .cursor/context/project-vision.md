# Project Vision: Personal Finance MVP

## Goal
A lean, responsive personal finance app for tracking budget and investments.

## MVP Features
1. **Budget Tracker**: Manual entry for Income/Expense, Categorization, Monthly Summaries.
2. **Investment Tracker**: Manual entry for Stocks/ETF/Crypto, P&L calculations, Asset Allocation.
3. **Dashboard**: Visual charts (Pie/Line) using Shadcn/Recharts.

## Technical Constraints
- **Database**: PostgreSQL (via Supabase or local).
- **Mobile First**: Must be fully responsive (PWA-ready).
- **Manual Input**: No API integrations for now. Everything is user-provided.
