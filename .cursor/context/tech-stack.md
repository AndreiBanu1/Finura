# Tech Stack & Tools

## Frameworks
- **Frontend**: Next.js 14+ (App Router), TypeScript.
- **Styling**: Tailwind CSS.
- **UI Components**: Shadcn UI (Radix UI).
- **Icons**: Lucide React.

## Data & Logic
- **ORM**: Prisma or Drizzle (use Drizzle if preferred).
- **Database**: PostgreSQL.
- **State Management**: React Context or simple URL state (keep it light).
- **Charts**: Recharts or Tremor.

## Deployment
- **Platform**: Vercel.
- **PWA**: next-pwa for "Add to Home Screen" functionality.
